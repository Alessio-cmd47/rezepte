# Einkaufsliste-Basis

Wiederkehrende Zutaten aus den 23 Rezepten. Was du **immer** im Haus haben solltest, damit du nur noch die rezeptspezifischen Sachen dazu kaufen musst.

Zahl in Klammern = Anzahl Rezepte, in denen die Zutat vorkommt.

## REZEPTSPEZIFISCH (nur kaufen wenn das Rezept dran ist)

- Garam Masala / Kurkuma → Butter Chicken
- Sesamöl + Soja + Reisessig → Egg Roll Bowl
- Mandelmehl/Hafermehl/Erdnussbutter/Chia/Kirschen → Cookie Cups
- Coleslaw-Mix + Mandeln + BBQ → Hot Honey BBQ
- Maultaschen + More Nutrition Gewürze → Maultaschensalat
- Mandelmilch + More Protein → Protein-Eis
- Backkakao + Erythrit + Magerquark → Brownie-Cheesecake
- Bens Express Reis → Feta-Reis Airfryer
